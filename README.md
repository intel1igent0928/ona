# Ona Website

This is a static website project for the Ona landing pages, built with HTML, CSS, and JavaScript.

## Project structure

- `index.html` — main landing page
- `about.html`, `contact.html`, `programs.html`, `donate.html`, etc. — project sections and pages
- `css/` — styles
- `sass/` — Sass sources
- `img/` — images
- `fonts/` — custom fonts
- `nav.js`, `script.js` — site scripts

## How to use

1. Open `index.html` in your browser.
2. Or run a local web server for better testing.

### Local server example

```bash
cd /d/Abdulboriy/Portfoio/ona
python -m http.server 8000
```

Then open `http://localhost:8000`.
